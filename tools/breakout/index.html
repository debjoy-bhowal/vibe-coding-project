How to Play Breakout Arena

🎯 Objective

Break all colored bricks using the ball and paddle. Complete all levels to win!

🎮 Controls

  • • Mouse Movement: Control paddle position
  • • Touch/Drag: Mobile paddle control
  • • Space/Click: Launch ball at game start

🧱 Brick Types

Red - Basic brick (10 pts)
Blue - Standard brick (20 pts)
Green - Strong brick (30 pts)
Yellow - Power brick (50 pts)
Purple - Bonus brick (40 pts)
Gray - Unbreakable

⚡ Power-ups

Expand - Wider paddle
Multi-ball - Extra balls
Fireball - Passes through all bricks
Slow-Mo - Slower ball speed

🏆 Scoring & Features

  • • Combo System: Consecutive brick hits multiply score
  • • Moving Bricks: Some bricks float and move around
  • • Multi-ball Action: Multiple balls for faster clearing
  • • Power-up Strategy: Collect and combine power-ups
  • • Progressive Difficulty: Each level gets more challenging

💡 Pro Tips

  • • Aim for corners and edges to create interesting ball angles
  • • Collect power-ups quickly before they fall off screen
  • • Use fireball to break through tough brick formations
  • • Save multi-ball power-ups for dense brick areas
  • • Build combos by hitting multiple bricks quickly
