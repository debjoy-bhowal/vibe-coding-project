## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required - runs entirely in the browser

### Usage
1. Open `index.html` in your web browser
2. Select your canvas size from the dropdown (16x16 recommended for beginners)
3. Choose a color from the palette or use the color picker
4. Select a drawing tool (Paint, Erase, or Fill)
5. Click and drag on the canvas to create your pixel art
6. Use the export button to save your creation
